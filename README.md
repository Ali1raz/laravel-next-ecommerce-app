# Ecommerce Store
Ecommerce website built using Laravel backend and Next.js, nodemailer and axios, tailwind CSS and Shadcn UI.

Features:
- Admin dashboard
- Seller Dashboard
- Buyers Dashboard
- Login and Register with forgot password and email verification.
- Email Verification


- Admin can manage all products and users, update user roles.
- Admins and sellers can manage products.
- Sellers can only see products and add to cart.
- ... and of course **checkout**.


More inside [this](https://github.com/Ali1raz/laravel-next-ecommerce-app/tree/master/shopping-frontend) directory.
